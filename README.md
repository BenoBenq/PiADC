# PiADC

To run this with a Raspberry Pi:
Connect the ADC to the SPI pins on the Pi and connect the LED to GPIO pin 5.
Change IP address in pc.c file and run make.

To run the program execute the following programs.
On the Pi:
```
./dataAq
```
```
sudo ./pi
```
On the local pc:
```
./pc on local pc
```
